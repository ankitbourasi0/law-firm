## Law Firm Website⚖️
### It is a simple minimal ui website for a law firm, with some smooth animation, click to scroll button, server side and statically side rendering. also has cool blog section where owner can publish 
* Blog Post
* Authors
* Categories,
* and More   

## Tech stack
### Next.Js
### GraphQL
### Hygraph CMS
### Framer
### Tailwind Css
